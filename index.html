MutationObserver HTMLStyleSheet Bug Report

Browser Information:

Test Description

This test attempts to use MutationObserver to monitor changes to a HTMLStyleSheet's content using { childList: true } options.

Expected behavior: MutationObserver should detect when CSS rules are added, removed, or modified in the stylesheet.

Observed behavior in Firefox: MutationObserver does not fire when stylesheet content changes.
